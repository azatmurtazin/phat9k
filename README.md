# Phat9k - PHP code analyzer, interpreter and transpiler

A Go-based PHP code parser, analyzer, interpreter, and transpiler that converts PHP code to Go code.

## Features

- **Parser** - Parse PHP code into an AST
- **Analyzer** - Static analysis with type inference
- **Interpreter** - Execute PHP code directly
- **Transpiler** - Convert PHP to Go
